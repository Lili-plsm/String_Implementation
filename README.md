Реализована функция sprintf для форматированного вывода в строку.

Поддерживаются спецификаторы: %c, %d, %f, %s, %u, %%.

Реализованы флаги -, +, пробел; ширина поля (number); точность .number; модификаторы длины h, l.

Добавлены спецификаторы %g, %G, %e, %E, %x, %X, %o, %p, флаги #, 0, ширина и точность через */.*, длина L.

Функция полностью протестирована юнит-тестами, проверка совпадения с библиотечной реализацией.

Реализованы стандартные функции string.h (memcpy, strcpy, strcat, strlen и др.)

Добавлены специальные функции обработки строк: to_upper, to_lower, insert, trim.
